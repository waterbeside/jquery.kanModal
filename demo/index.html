kanModal

这是一个简单的弹出模态框插件，基于bootstrap

此原为sco.modal.js项目，但因原作者多年没更新，已不适用现行的bootstrap版本。于是只好在原基础上进行修改，以适应现行bootstrap版本。后又为了满足一些个人项目的需求，又陆陆续续地进行多次修改和添加功能，如使modal可以拖动，加入了底部默认按钮，添加load，reload方法等等

一·用法

demo
弹出模态框 1 弹出模态框 2

二·设置

名称说明
title 标题
remote 使用ajax加载内容时的url
target 生成的模态框指定id，默认为#modal
draggable 是否可拖放
handle 如果能拖放，指定鼠标能拖放对像，默认为.modal-header
width 指定宽度
btns 设定按钮，默认有['refresh','close','submit']三个按钮
data ajax请求时设定传递的参数
content 如果不使用ajax加载内容，可以用data设置指定内容

三·事件

事件参数说明
onLoadSuccess self 数据加载完成时触发
onClose options 当关闭模态框时触发

四·方法

方法参数说明
show null 显示模态框
close null 关闭模态框
reload null 重载数据
load url 加载指定地址数据
